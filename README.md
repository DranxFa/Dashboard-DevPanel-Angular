# DevPanel

Panel de administración construido para practicar Angular 21.

## Tecnologías
- Angular 21 (Standalone, Signals, Reactive Forms)
- Tailwind CSS
- TypeScript

## Instalación
```bash
git clone https://github.com/DranxFa/Dashboard-DevPanel-Angular.git
cd devpanel
npm install
ng serve
```
